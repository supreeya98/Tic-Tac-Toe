# Tic Tac Toe Web App

A modern, interactive Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- 🎮 Interactive 3x3 game board
- 🎯 Two-player local gameplay (X and O)
- 🏆 Win detection for rows, columns, and diagonals
- 🤝 Draw detection
- ✨ Modern visual design with animations
- 📱 Responsive design for all devices
- 🔄 Reset functionality

## How to Play

1. Open `index.html` in any web browser
2. Player X goes first
3. Click on any empty cell to place your mark
4. Get three marks in a row (horizontal, vertical, or diagonal) to win
5. Click "Reset Game" to start a new game
